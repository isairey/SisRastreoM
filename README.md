# 📡🌍 Sistema de Rastreo en Tiempo Real

Aplicación web desarrollada con **JavaScript, HTML y CSS** para el monitoreo y visualización de ubicaciones en tiempo real, permitiendo rastrear dispositivos, personas o vehículos desde el navegador.

---

## 📌 Descripción

El **Sistema de Rastreo en Tiempo Real** es una solución ligera y accesible que permite obtener, mostrar y gestionar datos de geolocalización directamente desde el navegador.

Utiliza las capacidades de geolocalización del cliente o fuentes externas para representar ubicaciones en un entorno visual claro e interactivo.

---

## 🚀 Características

- 📍 Obtención de ubicación en tiempo real  
- 🗺️ Visualización en mapa (opcional)  
- 📊 Historial de ubicaciones  
- ⚡ Actualización dinámica  
- 🔐 Gestión básica de usuarios (opcional)  
- 📱 Diseño responsive  

---

## 🛠️ Tecnologías utilizadas

- **JavaScript (ES6+)** → Lógica de la aplicación  
- **HTML5** → Estructura  
- **CSS3** → Diseño y estilos  

### Integraciones (opcional)
- **Leaflet / Google Maps API** → Mapas interactivos  
- **API Geolocation** → Ubicación del navegador

## ▶️ Uso

- Permitir acceso a la ubicación
- Visualizar posición actual
- Consultar ubicaciones registradas
- (Opcional) Mostrar en mapa interactivo

---

💡 Funcionamiento

El sistema utiliza la API de Geolocalización del navegador para obtener coordenadas del usuario:

- 📍 Obtiene latitud y longitud
- ⚡ Actualiza la información en tiempo real
- 🗺️ Muestra la ubicación en la interfaz
- 💾 Puede almacenar datos localmente o en servidor
